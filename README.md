VatanBetaV3.0

--Vatan oyunumuz tarihteki kritik figürlerin aldığı kararları kendi tercihimize göre alıp kararlarımızın sonucunda neler olabileceği hakkında tarihsel temellere dayanan bir Strateji tabanlı tarih oyunudur---

1. Vatan.exe'yi açınız...
2. Windows bilgisayarınızı korudu uyarısı verirse eğer program ek bilgiye tıklayıp izin vere tıklayın...
3. Eğer bilgisayaranızda Java 10 dan sonraki sürüm yoksa hata verecek ve Java 25 kurulum sayfasına yönlendirecektir...
4. Açılan ekranda;
   \1. Chapter 1:Şehzade Mustafa(Henüz Çıkış Yapmadı)
   \2. Chapter 2:II.Abdulhamid(Henüz Çıkış Yapmadı)
   \3. Chapter 3:Enver Paşa
   \4. Chapter 4:Kazım Karabekir(Henüz Çıkış Yapmadı)
   \5. Chapter 5:Osman Pamukoğlu
  seçeneklerinden birini seçin ve aldığınız kararlara göre ülkenin geleceğini belirleyin... İyi Oyunlar!
